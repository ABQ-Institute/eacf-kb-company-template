# 1. Company

Organisation-wide context. The single source of truth for who you are, how you're structured, where you're going, and how you operate.

## Structure

| Folder | Purpose |
|--------|---------|
| `1-identity/` | Mission, vision, values, positioning |
| `2-structure/` | Org chart, founders, advisors |
| `3-strategy/` | Current focus, OKRs, roadmap |
| `4-policies/` | Communication, security, finance |
| `5-brand/` | Voice & tone, visual identity |
| `6-decisions/` | Architecture Decision Records (company-level) |

## Start Here

Begin with `1-identity/1-mission-vision.md` — it anchors everything else.

## Rules

- All content here is organisation-wide truth. Accessible to all agents and team members.
- Changes require approval from the company-layer owner (set in `0-meta/kb-config.yaml`).
- Keep it current. Stale company context is worse than no context.
